Install Gauge

Linux and Mac OS X tarballs

Download the tarball and execute: {% highlight bash %} ~ $ tar -xvf /usr/local gauge-$VERSION-$OS.$ARCH.tar.gz {% endhighlight %}

Choose the archive file appropriate for your installation. For instance, if you are installing gauge version {{ site.data.version.current }} for 64-bit x86 on Linux, the archive you want is called gauge-{{ site.data.version.current }}-linux.x86_64.tar.gz. Usually these commands must be run as root or through sudo.

Windows

The Gauge project provides installer for Windows users (besides installing from source). The installer configures your installation automatically and add gauge executable into the PATH environment variable.

If installation directory is not %PROGRAMFILES%, then you need to set GAUGE_ROOT environment variable manually after installation. GAUGE_ROOT should be set as the path where Gauge is installed.
