Get Started

Gauge is super simple to install. The section below will show you how to get Gauge working in 4 simple steps. We think it will take you 5 mins. That’s how easy it is!

  • Macintosh
  • Windows
  • Linux

  • There are multiple ways to get started with Gauge. You can read our detailed documentation for other methods of installation. Here we show you a simple 4 step process to get you running with Gauge on a mac.

    1. Install Gauge

    You can download the pkg and follow these instructions to install Gauge
    32 bit 64 bit
    Mac OS X.pkg Show SHA1
    Or we would recommend using brew for a very simple installation using this command.
    ~ $ gauge specs/hello_world.spec 
    2 scenarios executed, 0 failed
1 specifications executed, 0 failed

    2. Install Core Plugins

    Gauge has a rich plugin repository. Language runners, IDEs, reports are all plugin. However, since Gauge is a first class command line tool, you only *need* one plugin to get your Gauge tests running. That is your language runner plugin.
    ~ $ gauge specs/hello_world.spec 
    2 scenarios executed, 0 failed
1 specifications executed, 0 failed
    Go here to see how to install other language runners and plugins

    3. Test your installation

    Run this command to check if all plugins are installed and are at the latest version
    ~ $ gauge specs/hello_world.spec 
    2 scenarios executed, 0 failed
1 specifications executed, 0 failed

    4. Execute Gauge specs

    Execute the hello world specs so you know Gauge is working!
    ~ $ gauge specs/hello_world.spec 
    2 scenarios executed, 0 failed
1 specifications executed, 0 failed
    That’s it. You’re ready to start testing!
