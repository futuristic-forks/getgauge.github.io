Features

First class command line support
Do everything with your terminal, well almost everything. Gauge allows you to perform a variety of operations via the command line.
Multiple language support
Write test cases in your favorite programming language.
Simple Syntax
Author specifications in markdown, a widely-known markup syntax.
Parallel Execution
Simplest Parallelization strategy across all the languages.
Extensible and Hackable
Get more out of Gauge via its plugin architecture. Extension points include: language support, driver support, test execution events, and more.
Multiple IDE Support
Gauge is designed to support IDE integration. Intellij idea integration helps to collaborate better by providing smart suggestions.
